Play [TIC-TAC-TOE](https://tictac-ef8e4.web.app/).

## Available Scripts

This is 2 player tic-tac-toe game, where 2 people can play from different computers or browser windows

## Stack

1. React
2. TypeScript
3. Node
4. Firebase
5. Semantic-UI


## Run locally

1. Clone the repo
2. run 'npm install'
3. run 'npm start'
